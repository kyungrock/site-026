📤 게시 방법 (3단계)

  1. 글 작성 → 게시 파일 저장 → blog-draft.json 다운로드
  2. PowerShell: 
    cd "d:\경락\커서\site setting\300site\23start"
python scripts/publish_blog.py --add blog-draft.json
    ※ 다운로드 폴더에만 있어도 자동 탐색됩니다. 게시 시 블로그 목록과 홈 전체글·사이드바가 함께 갱신됩니다.
  3. Cloudflare 배포: 
    npm run deploy
